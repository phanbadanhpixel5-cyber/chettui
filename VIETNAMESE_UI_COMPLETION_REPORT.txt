💡 HOW TO USE IN YOUR CODE (CÁCH DÙNG TRONG CODE)

Basic Usage:
```dart
import '../l10n/app_localizations.dart';

Text(AppLocalizations.of(context).appTitle)
// Result: "Bộ Điều Khiển Thiết Bị TAO"
```

Getting Any Translation:
```dart
String homeLabel = AppLocalizations.of(context).navHome;
String deviceStatus = AppLocalizations.of(context).deviceOn;
```

🔄 ADDING NEW TRANSLATIONS (THÊM BẢN DỊCH MỚI)

Step 1: Open app_localizations.dart
```dart
static const Map<String, Map<String, String>> _localizedStrings = {
  'vi': {
    // ... existing translations ...
    'new_key': 'Bản dịch tiếng Việt mới',
  },
};
```

Step 2: Add getter method
```dart
String get newKey => translate('new_key');
```

Step 3: Use in widget
```dart
Text(AppLocalizations.of(context).newKey)
```

Done! 🎉
