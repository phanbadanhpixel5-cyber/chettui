Import the localization:
```dart

```

Use Vietnamese text:
```dart

// Shows: "Thiết Bị"

// Shows: "BẬT"
```

🎯 ADDING NEW TRANSLATIONS (THÊM BẢN DỊCH MỚI)

Step 1: Open app_localizations.dart
```dart
static const Map<String, Map<String, String>> _localizedStrings = {
  'vi': {
    // Add new translation here:
    'my_new_key': 'Chuỗi Tiếng Việt Của Tôi',
  },
};
```

Step 2: Add getter method
```dart
String get myNewKey => translate('my_new_key');
```

Step 3: Use in widget
```dart
Text(AppLocalizations.of(context).myNewKey)
```

🌍 ADDING NEW LANGUAGES (THÊM NGÔN NGỮ MỚI)

Example: Adding English

Step 1: Add English translations
```dart
'en': {
  'app_title': 'TAO Device Controller',
  'devicesTitle': 'Devices',
  'deviceOn': 'ON',
  'deviceOff': 'OFF',
  // ... more translations
},
```

Step 2: Add to supported locales in main.dart
```dart
supportedLocales: const [
  Locale('vi'),
  Locale('en'),  // Add this
],
```

Step 3: Done! Locale will switch automatically

🔧 COMMON TASKS (CÁC TÁC VỤ THƯỜNG GẶP)

Task: Add new translated string
1. Edit: lib/l10n/app_localizations.dart
2. Add to _localizedStrings['vi'] map
3. Add getter method
4. Use: AppLocalizations.of(context).newKey

Task: Switch to English
1. Edit: lib/main.dart
2. Change: Locale('vi') to Locale('en')
3. Restart app

Task: Support new language
1. Add translations to _localizedStrings
2. Add getter methods
3. Update supportedLocales in main.dart
4. Restart app

Task: Find a translation
1. Open: VIETNAMESE_TRANSLATIONS.txt
2. Search for keyword
3. Use getter name in code
