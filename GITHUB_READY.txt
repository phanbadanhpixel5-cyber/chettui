❓ FREQUENTLY ASKED QUESTIONS
────────────────────────────────────────────────────────────────────────────

Q: What if I see "fatal: remote origin already exists"?
A: The script will ask to replace it. Say yes (y) to use your new URL.

Q: Do I need to create .env files?
A: No! .env.example provides a template. Users copy it and fill in values.

Q: Can I make the repository private?
A: Yes! When creating on GitHub, choose "Private" instead of "Public".

Q: What if authentication fails?
A: See "Troubleshooting" in GITHUB_SETUP_GUIDE.md for SSH/Token setup.

Q: Can I change the repository name later?
A: Yes! GitHub Settings → Rename Repository

Q: How do I update my repository after making changes?
A: git add . && git commit -m "message" && git push

Q: Can others clone my repository?
A: Yes! If public. They run: git clone YOUR_REPO_URL

Q: How do I add team members?
A: GitHub Settings → Collaborators → Add by username/email
