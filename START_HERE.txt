⚡ QUICK START (30 minutes)
─────────────────────────────────────────────────────────────────────────────

Step 1: Backend (5 min)
  cd backend
  npm install
  cp .env.example .env
  npm start

Step 2: Arduino (10 min)
  1. Load arduino/DeviceController.ino in Arduino IDE
  2. Select board and COM port
  3. Upload sketch
  4. Verify with Serial Monitor

Step 3: Flutter (5 min)
  cd flutter
  flutter pub get
  flutter run

Step 4: Configure (10 min)
  • Set backend IP in device_provider.dart
  • Add Firebase credentials to .env
  • Set OpenWeatherMap API key
  • Update Arduino COM port
