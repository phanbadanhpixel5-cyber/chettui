=== USAGE EXAMPLE ===
Widget build(BuildContext context) {
  final l10n = AppLocalizations.of(context);
  return Text(l10n.appTitle);  // "Bộ Điều Khiển Thiết Bị TAO"
}

=== HOW TO ADD NEW STRINGS ===
1. Open lib/l10n/app_localizations.dart
2. Add to _localizedStrings['vi'] map:
   'new_key': 'Chuỗi tiếng Việt',
3. Add getter method:
   String get newKey => translate('new_key');
4. Use in widgets:
   Text(AppLocalizations.of(context).newKey)
