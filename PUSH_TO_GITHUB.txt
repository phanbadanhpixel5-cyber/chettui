❓ TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────

Q: Error "remote origin already exists"
A: Run: git remote remove origin
   Then add the remote again

Q: Error "authentication failed"
A: Make sure you:
   • Have GitHub account
   • Generated Personal Access Token OR configured SSH
   • Typed correct credentials

Q: Error "Permission denied"
A: If using SSH:
   • Generate SSH key: ssh-keygen -t ed25519
   • Add to GitHub: Settings → SSH and GPG keys
   • Paste public key content

Q: Files not showing up
A: Check .gitignore - file might be excluded
   Run: git status (to see what will be pushed)

Q: Want to change repository URL
A: git remote set-url origin https://new-url.git
   Or: git remote remove origin
       git remote add origin https://new-url.git
